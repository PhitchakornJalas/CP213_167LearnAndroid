# ✈️ Travel Savings Planner (แอปพลิเคชันวางแผนออมเงินเพื่อการเดินทาง)

แอปพลิเคชันที่ช่วยให้การวางแผนการเดินทางของคุณเป็นเรื่องง่ายและมีระเบียบวินัยทางการเงิน ด้วยระบบการวางแผนกิจกรรมควบคู่ไปกับการตั้งเป้าหมายออมเงินอัจฉริยะ

## 🌟 ฟีเจอร์หลัก (Key Features)

### 1. วางแผนกิจกรรมการเดินทาง (Travel Planning)
*   สร้างกิจกรรมรายวัน กำหนดเวลา (Check-in/Check-out) หรือกิจกรรมตลอดวัน
*   ระบบตรวจสอบเวลาทับกัน (Time Overlap Check) ป้องกันการจัดตารางซ้อน
*   แนบงบประมาณแยกตามหมวดหมู่: ค่าเดินทาง, ค่าอาหาร, ค่าที่พัก, ค่าช้อปปิ้ง และอื่นๆ

### 2. ระบบออมเงินอัจฉริยะ (Smart Savings System)
*   **Dynamic Redistribution**: คำนวณยอดออมต่อวันให้อัตโนมัติ โดยอ้างอิงจาก `(งบประมาณที่เหลือ / จำนวนวันที่เหลือ)` หากมีการเพิ่มงบหรือออมขาดไป ระบบจะช่วยเฉลี่ยยอดใหม่ให้ทันที
*   **Sequential Locking**: สร้างวินัยการออมด้วยระบบล็อคการออมของวันนี้ หากยอดออมของวันก่อนหน้ายังไม่ครบถ้วน
*   **Flexible Saving Plans**: เลือกเริ่มออมได้หลากหลายรูปแบบ ทั้ง "ตั้งแต่วันนี้" หรือกำหนดระยะเวลาล่วงหน้าเป็นวัน สัปดาห์ เดือน หรือปี

### 3. การจัดการการเงินและหลักฐาน (Financial Management)
*   **QR Payment**: รองรับการสร้าง QR Code PromptPay เพื่อออมเงินเข้าบัญชีเป้าหมายได้ทันที
*   **Slip Verification**: ระบบตรวจสอบสลิปการโอนเงินอัตโนมัติ เพื่อป้องกันการออมซ้ำและยืนยันยอดเงินจริง
*   **Progress Tracking**: ดูความคืบหน้าการออมแยกตามรายกิจกรรมและภาพรวมทั้งทริป

### 4. ระบบแจ้งเตือน (Notification System)
*   **Countdown Alerts**: แจ้งเตือนนับถอยหลังก่อนวันเดินทาง (เช่น 3 วันก่อนไป)
*   **Saving Reminders**: แจ้งเตือนให้ไม่พลาดการออมเงินประจำวัน
*   **Event Reminders**: แจ้งเตือนล่วงหน้าก่อนเริ่มกิจกรรม (1, 2 หรือ 3 ชั่วโมง)
*   **Immediate Alerts**: แจ้งเตือนทันทีที่สร้างกิจกรรมใหม่หากต้องเริ่มออมตั้งแต่วันนี้

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)
*   **Framework**: Flutter (Dart)
*   **Backend**: Firebase (Firestore, Authentication)
*   **State Management**: Provider
*   **Local Storage**: SharedPreferences (สำหรับตั้งค่า Notification)
*   **Notifications**: Flutter Local Notifications
*   **Date & Time**: Intl (รองรับภาษาไทยเต็มรูปแบบ)

## 📸 ตัวอย่างการใช้งาน
*   หน้าปฏิทิน: ดูภาพรวมกิจกรรมและสถานะการออมรายวัน
*   หน้าเพิ่มกิจกรรม: กำหนดแผนการออมเงินที่ล็อคอัตโนมัติเมื่อเริ่มใช้งานเพื่อความแม่นยำของข้อมูล
*   หน้ารายการออม: ตรวจสอบและกดออมเงินพร้อมแนบสลิป
